### Mochila pequeña

<img src="https://raw.githubusercontent.com/divisionind/BackpacksRemastered/master/screenshots/backpack-small.png">

### Mochila grande

Una mochila bastante grande. Del tamaño de un cofre doble.

<img src="https://raw.githubusercontent.com/divisionind/BackpacksRemastered/master/screenshots/backpack-large.png">

### Mochila enlazada

Una mochila capaz de interactuar con muchos tipos de contenedores: cofres, cofres dobles, hornos, etc. Funciona a cualquier distancia, incluso a través de las dimensiones. Es la mochila definitiva. Para enlazarla, sujétala con la mano y haz clic con el botón izquierdo del ratón en el contenedor deseado.

<img src="https://raw.githubusercontent.com/divisionind/BackpacksRemastered/master/screenshots/backpack-linked.png">

### Mochila combinada

Mochila que puede guardar otras mochilas para uso rapido.

Para añadir una mochila, pon la mochila combinada junto con la mochila-que-se-insertará en la cuadrilla de crafteo.

Para remover una mochila, mirar: [/bp split]().

<img src="https://raw.githubusercontent.com/divisionind/BackpacksRemastered/master/screenshots/backpack-combined.png">

## Utilizar una mochila

Para usar una mochila, primero tienes que craftear la mochila que quieres usar y una [llave de mochila](#llave-de-la-mochila).

Luego tienes que ponerte la mochila en el pecho. Despues selecciona la llave de la mochila y dale al click derecho/boton de interacción.

## Comandos

Para saber todos los comandos disponibles, utiliza el comando de ayuda:

```shell
/bp help
```

### Lista de comandos utiles

#### /bp split

Separa una mochila que tengas en la mano del objeto con el que se ha combinado, o saca una mochila que se añadió a una mochila combinada.

Se executa de la siguiente manera:

```shell
/bp split
```

#### /bp info

Muestra información basica sobre el plugin.

Se executa de la siguiente manera:

```shell
/bp info
```
